# Droid-Control-Sim (Educational Simulation)

هذا المشروع عبارة عن بيئة محاكاة تعليمية تهدف لفهم كيفية التحكم في أجهزة الأندرويد برمجياً باستخدام **ADB**.

## الغرض من المشروع
* تعلم أوامر نظام الأندرويد العميقة.
* فهم كيفية إدارة الأجهزة عن بُعد.
* دراسة الثغرات الأمنية وكيفية الحماية منها في بيئة معزولة (Sandbox).

## المتطلبات
1. تثبيت **Python 3.x**.
2. تثبيت **ADB Platform Tools**.
3. وجود محاكي أندرويد (Android Emulator) مفعل عليه خيار `USB Debugging`.

## كيفية الاستخدام
قم بتشغيل المحاكي أولاً، ثم نفذ الأمر:
`python main.py`
